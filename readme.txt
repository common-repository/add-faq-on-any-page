== Description ==
"Add FAQ on any page" helps you to create dynamic FAQ for user website. You can add "Add FAQ on any page" in website posts, page and custom post type page. 

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. After that go to your plug-in manager and activate it.
3. On activation, "Add FAQ on any page" - Go to FAQ menu and create question and answer. 
4. Use [add_faq_onp] shortcode in your posts, pages and custom post type page to show your FAQ.
